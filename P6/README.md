# Praktikum Basis Data 2 - Database Apotek

## Deskripsi
Project ini merupakan praktikum Basis Data 2 tentang penerapan query SQL dan subquery pada database apotek menggunakan MySQL.

## Materi Praktikum
- Membuat database dan tabel
- Insert dan menampilkan data
- Query dasar SQL:
  - SELECT
  - WHERE
  - LIKE
  - ORDER BY
  - JOIN
  - COUNT, SUM, AVG
  - GROUP BY
- Penerapan Subquery SQL

## Kesimpulan
Praktikum ini membantu memahami cara membuat dan mengelola database relasional menggunakan SQL. Selain query dasar, penggunaan subquery mempermudah pengolahan data yang lebih kompleks seperti mencari data terbesar, terkecil, dan rata-rata pada database apotek.
